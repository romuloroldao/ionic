O que é ionic?


Ionic Framework é uma SDK
open source que permite desenvolvedores a contruir aplicativos mobiles de altissima qualidade utilizando linguagens de tecnologia web como por exemplo HTML,CSS e JavaScript

O que é IONIC Cloud?

O Ionic Cloud é uma plataforma para gerenciar e hospedar aplicativos Ionic, com alguns serviços integrados como por exemplo notificações por push, construções nativas, autentificação de usuario e atualisações ao vivo.
